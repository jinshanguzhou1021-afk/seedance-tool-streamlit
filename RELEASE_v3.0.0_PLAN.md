## 🎯 功能特性

### ✅ 视听一致性引擎

| 层级 | 功能 |
|------|------|
| **全局逻辑层** | 核心角色设定、环境基调锁定、光影风格统一 |
| **导演编剧层** | 镜号管理、场景定义、运镜设计、旁白+音效 |
| **Seedance 适配层** | 高张力、细腻表情、物理交互、完整参数 |

### ✅ 标准化剧本输出

每镜包含 6 要素：
1. 镜号 (Shot ID) - 如：Shot 01
2. 场景 (Setting) - 如：破旧小酒馆 - 深夜
3. 运镜 (Camera) - 如：低角度推近特写
4. 动态画面描述提示词 (Visual Prompt) - 完整的 Seedance 2.0 提示词
5. 旁白 (VO) - 叙事台词或内心独白
6. 音效 (SFX) - 环境音、动作音、情绪转场音

### ✅ 顶级系统 Prompt

系统 Prompt 包含：
- 顶级电影导演与 AI 视频专家的角色定义
- 空间连续性保证机制
- 角色一致性保证机制
- Seedance 2.0 深度优化指令
- 标准化输出格式要求

## 💡 最佳实践

### 故事输入建议

**推荐长度**：100-300 字

**包含元素**：
- 核心角色（外貌、服装、性格）
- 核心环境（时间、地点、天气）
- 核心冲突或转折
- 整体风格基调

**示例**：
```
完整版（100-300字）：
在极寒的雪原夜晚，一名身着黑色斗篷的复仇者孤独地前行。他手持一把泛着寒光的古剑，剑刃上满是陈年的血痕。前方是一座破败的古城，城墙在风雪中若隐若现。复仇者停下脚步，凝视着城门，眼中燃烧着复仇的火焰。他猛然拔剑，剑尖指向夜空，发出无声的嘶吼。
```

### 全局风格锁定建议

**推荐在系统 Prompt 中锁定**：

1. **视觉风格**：
   - 赛博朋克：霓虹、冷蓝、未来科技
   - 东方武侠：水墨、留白、金色、古雅
   - 史诗战争：宏大、悲壮、史诗配乐

2. **光影基调**：
   - 高对比度（强张力）
   - 柔和光效（治愈感）
   - 强烈明暗（戏剧化）

3. **角色特征锁定**：
   - 发型、发色
   - 服装风格
   - 面部特征
