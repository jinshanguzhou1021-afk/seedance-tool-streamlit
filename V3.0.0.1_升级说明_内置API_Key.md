---

### 方式 2：使用环境变量（推荐生产环境）

```bash
# 设置环境变量
export DEEPSEEK_API_KEY="your_api_key_here"

# 启动应用
streamlit run app.py
```

**优先级**：环境变量 > 内置配置

---

### 方式 3：使用 Streamlit Secrets（推荐生产环境）

创建 `.streamlit/secrets.toml` 文件：

```toml
[api]
deepseek_api_key = "your_api_key_here"
openai_api_key = "your_openai_key_here"
openai_base_url = "https://api.openai.com/v1"
```

**优先级**：Secrets > 环境变量 > 内置配置

## ❓ 常见问题

**Q: API Key 是否安全？**
A: 内置 API Key 仅用于个人学习和测试，生产环境建议使用 Secrets 或环境变量。

**Q: 如何更换 API Key？**
A: 可以在 `app.py` 的第 158 行修改 `DEEPSEEK_API_KEY`，或者使用环境变量覆盖。

**Q: 支持哪些 API 模式？**
A: 目前支持 DeepSeek-V3（免费）和 OpenAI GPT-4（付费）。

**Q: 如何查看 API 使用情况？**
A: 可以登录 DeepSeek 或 OpenAI 的官方网站查看。
