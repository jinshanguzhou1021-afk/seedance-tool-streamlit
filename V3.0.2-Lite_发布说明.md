## 🔑 核心特性

### 1. API 安全配置

**三层优先级**：
1. ✅ **st.secrets**（生产环境推荐）
2. ✅ **环境变量**（开发环境）
3. ⚠️ **内置 fallback**（仅用于本地测试）

**代码实现**：
```python
try:
    API_KEY = st.secrets["DEEPSEEK_API_KEY"]
    API_MODE = "Secrets"
except (KeyError, FileNotFoundError):
    API_KEY = os.getenv("DEEPSEEK_API_KEY", "")
    if API_KEY:
        API_MODE = "环境变量"
    else:
        API_KEY = "sk-62a693ae5cb24574bd9df2a9bb53cd99"
        API_MODE = "内置 fallback"
        st.sidebar.warning("⚠️ 正在使用硬编码的 API Key，请尽快迁移到 st.secrets")
```

**优势**：
- ✅ 生产环境安全
- ✅ 开发环境方便
- ✅ 清晰的模式提示

### 2. JSON 强制输出

**关键代码**：
```python
response = client.chat.completions.create(
    model=API_MODEL,

    temperature=0.7,

result_json = json.loads(response.choices[0].message.content)
```

**优势**：

## 🚀 使用方式

### 快速开始

```bash
cd /workspace/projects/apps/seedance-tool-streamlit
streamlit run app_v3.0.2_lite.py
```

### 访问地址
```
http://localhost:8501
```

### 配置 API Key

**方式 1：使用内置 Key（默认）**
✅ 无需配置，直接使用

**方式 2：环境变量（推荐开发环境）**
```bash
export DEEPSEEK_API_KEY="your_api_key_here"
streamlit run app_v3.0.2_lite.py
```

**方式 3：Streamlit Secrets（推荐生产环境）**
创建 `.streamlit/secrets.toml`：
```toml
DEEPSEEK_API_KEY = "your_api_key_here"
```
