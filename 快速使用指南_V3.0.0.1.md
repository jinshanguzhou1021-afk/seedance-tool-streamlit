## 🎯 快速开始

### 1. 启动应用

```bash
cd /workspace/projects/apps/seedance-tool-streamlit
streamlit run app.py
```

访问地址：http://localhost:8501

## 📖 使用步骤

### AI 导演功能

#### 步骤 1：切换到 AI 导演控制台
在应用左侧选择"AI 导演控制台"

#### 步骤 2：开启 API
打开"🤖 使用 DeepSeek-V3 API"开关

**注意**：API Key 已内置，无需配置！

#### 步骤 3：输入故事
在"📖 1. 故事/脚本输入"中输入你的故事

**示例**：
```
在极寒的雪原夜晚，一名身着黑色斗篷的复仇者孤独地前行。
他手持一把泛着寒光的古剑，剑刃上满是陈年的血痕。
前方是一座破败的古城，城墙在风雪中若隐若现。
复仇者停下脚步，凝视着城门，眼中燃烧着复仇的火焰。
他猛然拔剑，剑尖指向夜空，发出无声的嘶吼。
```

#### 步骤 4：调整参数（可选）
- **张力强度**：3-10 级（默认：明显起伏（中等））
- **情绪滤镜**：忧郁、惊悚等
- **表情微操**：瞳孔微动、汗水滑落等
- **动作张力**：慢动作爆发、碎屑飞溅等
- **大师运镜**：希区柯克变焦、环绕运镜等
- **戏剧化光影**：高对比度、戏剧化光效等

#### 步骤 5：生成分镜
点击"🎬 生成 AI 全能导演级分镜"按钮

#### 步骤 6：查看结果
- 全局设定（角色、环境、光影）
- 分镜剧本（6 要素：镜号、场景、运镜、提示词、旁白、音效）
- 完整 Seedance 2.0 提示词

#### 步骤 7：下载
点击"💾 下载 v3.0.0 导演级分镜配置"按钮

## 🎨 功能说明

### 三层导演逻辑

#### 第一层：全局逻辑层（一致性保证）
- 核心角色设定（外貌、服装、性格）
- 环境设定（空间、时间、天气）
- 光影基调（色调、对比度、光效）

#### 第二层：导演编剧层（标准化剧本）
每个分镜包含 6 要素：
1. 镜号 (Shot ID)
2. 场景 (Setting)
3. 运镜 (Camera)
4. 动态画面描述提示词 (Visual Prompt)
5. 旁白 (VO)
6. 音效 (SFX)

#### 第三层：Seedance 2.0 适配层
- 高张力动态描述
- 细腻表情微操
- 物理交互效果

## 🔧 API 配置

### 方式 1：使用内置 API Key（默认）
✅ 无需配置，直接使用

### 方式 2：使用环境变量
```bash
export DEEPSEEK_API_KEY="your_api_key_here"
streamlit run app.py
```

### 方式 3：使用 Streamlit Secrets
创建 `.streamlit/secrets.toml` 文件：
```toml
[api]
deepseek_api_key = "your_api_key_here"
```

## 💡 最佳实践

### 故事输入建议
- **长度**：50-150 字
- **内容**：包含核心角色、环境、情绪转折
- **避免**：过于复杂或抽象的描述

### 张力强度选择
- **日常场景**：3-5 级
- **情感冲突**：5-7 级
- **动作场景**：7-9 级
- **史诗剧情**：10 级

### 情绪滤镜使用
- **忧郁**：冷色调，低饱和度，压抑氛围
- **惊悚**：高对比度，强烈明暗，锐利线条
- **温馨**：暖色调，柔和光效，舒适氛围
