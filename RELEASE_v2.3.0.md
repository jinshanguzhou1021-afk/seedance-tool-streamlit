## 🎯 使用指南

### 基础使用

1. **选择功能**：在侧边栏选择 "🎬 AI 视觉导演"
2. **输入故事**：在 "📖 故事/脚本输入" 中输入 100 字左右的故事或场景描述
3. **调节参数**：
   - 🎭 张力强度：选择整体张力（平淡→史诗）
   - 🎨 情绪滤镜：选择情绪基调（可多选）
   - 😐 表情微操：选择细腻表情（可多选）
   - 🏃 动作张力：选择爆发性动作（可多选）
   - 🎥 大师运镜：选择专业运镜（可多选）
   - 💡 戏剧化光影：选择光影效果（可多选）
   - 📐 画幅比例：选择视频画面比例
4. **生成提示词**：点击 "🎬 生成导演级分镜提示词"

### 高级技巧

**张力调节**：
- "平淡叙事"（motion=3）：适合平静、温和的场景
- "明显起伏"（motion=7）：适合有冲突、转折的场景
- "史诗冲突"（motion=10）：适合高潮、爆发的场景

**情绪组合**：
- 忧郁 + 恐惧 = 悬疑恐怖氛围
- 热血 + 动作 = 爆炸性动作场景
- 治愈 + 浪漫 = 温馨浪漫氛围
- 赛博 + 惊悚 = 赛博朋克悬疑场景

**示例输入**：
```
一个穿着黑色风衣的复仇者，在暴雨中的霓虹街道上追逐敌人。雨水顺着他紧锁的眉头和暴起的青筋滑落，他对着天空发出无声的嘶吼，背景雷光闪烁，强烈的明暗对比刻画出深沉的绝望。

张力强度：史诗冲突
情绪滤镜：赛博 + 悬疑
表情微操：愤怒 + 绝望
动作张力：剧烈喘息 + 肌肉颤抖
大师运镜：低角度仰拍 + 手持晃动感
戏剧化光影：强烈侧逆光 + 霓虹灯效 + 雷光闪烁
画幅比例：21:9 电影宽屏
```
